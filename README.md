# Discord OG Role & Viewer Bot

A powerful and easy-to-use Discord bot designed to:

* Automatically assign **OG roles** to early members
* Provide a **viewer system** to track and display OG members
* Offer simple commands for managing and viewing OG status

---

## 🚀 Features

* 🏆 OG role assignment system
* ⚡ Fast and lightweight
* 🔧 Easy setup and configuration
* 🧩 Prefix command support (!)

---

## 📜 Commands
(Administrator Required)

* `!setlimit 100`
  ➤ You can replace the 100 with how many og roles is given out ( it counts the role and how many people have it.)

* `!setroles @OG_ROLE_HERE @Role`
  ➤ Set the OG role & Role Given after the limit reached

## ⚙️ Installation

### 1. Clone the repository

```
git clone https://github.com/wmaontop/Discord-OG-Role-Viewer-Bot.git
cd Discord-OG-Role-Viewer-Bot
```

---

### 2. Install dependencies

```
npm install
```

---

### 3. Run the bot

```
node index.js
```

---

## 🛠 Requirements

* Node.js (v16 or higher)
* Discord bot token
* Discord server with admin permissions

---

## 📌 Notes

* Make sure the bot has permission to manage roles
* The OG role must be **below the bot role** in the role hierarchy
